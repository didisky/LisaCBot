## Project Overview

LisaCBot is a Java 25 Maven project.

## Build Commands

```bash
mvn compile          # Compile
mvn test             # Run tests
mvn package          # Build JAR
mvn clean install    # Full build
```

## Project Structure

Standard Maven layout:
- `src/main/java/com/lisacbot/` - Source code
- `src/main/resources/` - Resources
- `src/test/java/com/lisacbot/` - Tests
